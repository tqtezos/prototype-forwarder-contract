type LinearizeUStore a = GLinearizeUStore (Rep a) Source #

Get information about all fields of UStore template in a list.

In particular, this recursivelly traverses template and retrives names and types of fields. Semantic wrappers like UStoreFieldExt and |~> in field types are returned as-is.
