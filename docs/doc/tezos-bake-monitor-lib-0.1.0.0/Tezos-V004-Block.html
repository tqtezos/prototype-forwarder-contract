data WithProtocolHash a Source #

Simple wrapper for adding a protocol hash to some other value.

The Aeson instances are smart (too smart). If the underlying type is not an object or an object with a conflicting key with the one added by this type, the JSON encoding will be two-layered. Otherwise, the JSON encoding will simply add an additional key for the protocol information.
